## How To Use

#### Fast Installation

git clone the repo using
```html
git clone https://github.com/AmirMeiry/ShoppingListApp.git
```

enter the git folder and type
```html
npm install
```

this should install all the dependencies needed to run.

#### Installation

#### react installation
```html
    # NPM

    npm i react

    # Yarn

    yarn add react
```


#### axios installation
```html
    # NPM

    npm i axios

    # Yarn

    yarn add axios
```

#### redux installation
```html
    # NPM

    npm install redux

    # Yarn

    yarn add redux
```

#### react-redux installation
```html
    # NPM:

    npm install react-redux

    # Yarn:

    yarn add react-redux
```

#### redux-devtools-extension installation
```html
    # NPM:

    npm install --save redux-devtools-extension

    # Yarn:

    yarn add redux-devtools
```

#### redux-logger installation
```html
    # NPM

    npm i --save redux-logger

    # Yarn

    yarn add redux-logger
```

#### redux-thunk installation
```html
    # NPM

    npm install --save redux-thunk

    # Yarn

    yarn add redux-thunk
```

#### npm start
```html
    # Terminal / CMD

    npm start
```
